# Advanced Python Tic-Tac-Toe GUI

A **modern, graphical Tic-Tac-Toe game** built in Python using **Tkinter**.  
This project features an interactive GUI, score tracking, and winning line highlighting for a fun and polished experience.

---

## üéÆ Features

- Fully **graphical interface** (no terminal-based gameplay)  
- **Two-player mode** (Player X and Player O)  
- **Winning line highlight** when a player wins  
- **Scoreboard** to track the number of wins for each player  
- **Reset button** to start a new game without restarting  
- Clean and **modern UI design** using Tkinter  
- Easy to customize for themes or colors
